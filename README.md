# Finance Dashboard

A simple and interactive finance dashboard built using React. This application helps users track income, expenses, and overall financial activity in a clean and intuitive interface.

---

## Features

- View total balance, income, and expenses
- Add and delete transactions (Admin role)
- Role-based UI (Admin / Viewer)
- Filter and search transactions
- Data visualization using charts (Pie & Line)
- Spending insights (top category)
- Dark mode support
- Data persistence using localStorage

---

## Tech Stack

- React (Vite)
- Tailwind CSS
- Recharts

---

## Functionality

- Dashboard Overview: Displays financial summary with charts
- Transactions Section: Add, delete, filter, and search transactions
- Role-Based UI:
  - Admin → Full access
  - Viewer → Read-only access
- Insights Section: Shows top spending category

---

## How to Run

1. Clone the repository  
2. Run `npm install`  
3. Run `npm run dev`  

---

## Notes

- This is a frontend-only project using mock data
- Data is stored using localStorage
- Role-based behavior is simulated on the UI
